Donde Rebrota la Memoria

Brota la nueva vida: modelo de gestión urbana participativa para Villa Muñecas.

description Sobre el Proyecto

Esta propuesta aborda la situación de asentamientos vulnerables del AMeT, tomando Villa Muñecas como ejemplo. Este barrio fue identificado como el segundo más vulnerable de Argentina según TECHO (julio 2019).

Desarrolla un modelo innovador de gestión urbana integral que combina relocalización segura, espacios verdes, participación comunitaria y construcción colaborativa para transformar realidades urbanas.
